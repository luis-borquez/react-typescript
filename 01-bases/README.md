# Instrucciones para ejecutar el proyecto
1. Instalar dependencias `npm install` | `npm i`
2. En el archivo `src/main.ts` en la sección de "Temas" descomentar el tema que se desea ver
3. Iniciar el servidor de desarrollo `npm run dev`
